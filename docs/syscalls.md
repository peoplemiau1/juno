# Системные вызовы (Syscalls)

## Прямой доступ
Juno позволяет вызывать любые системные функции Linux напрямую через `syscall`. Поддерживается до 6 аргументов.
```juno
// Пример: mmap
let addr = syscall(9, 0, 4096, 3, 34, -1, 0)
```

## Регистры Syscall (Linux x86-64)
1.  Номер системного вызова: `RAX`
2.  Аргумент 1: `RDI`
3.  Аргумент 2: `RSI`
4.  Аргумент 3: `RDX`
5.  Аргумент 4: `R10`
6.  Аргумент 5: `R8`
7.  Аргумент 6: `R9`

## Обертки Syscall
Стандартная библиотека предоставляет удобные функции:
*   `fork()`, `execve(...)`
*   `getpid()`, `getuid()`
*   `mkdir(...)`, `rmdir(...)`, `unlink(...)`
*   `mmap(...)`, `munmap(...)`
*   `pipe(...)`, `dup2(...)`
