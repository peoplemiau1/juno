# Функции и Обобщения (Generics)

Функции в Juno являются строительными блоками логики. Они поддерживают аргументы, возвращаемые значения и обобщенные типы.

## Объявление функций

Основное ключевое слово — `fn`. Сигнатура может включать типы аргументов и возвращаемого значения через двоеточие.

```swift
fn add(a: int, b: int): int {
    return a + b
}
```

Если тип не указан, по умолчанию используется `int` (64-битное целое).

## Методы структур

Функции могут быть ассоциированы со структурами. В этом случае первым аргументом неявно или явно передается `self` (указатель на экземпляр структуры).

```swift
struct Rect {
    let w: int
    let h: int
}

fn Rect.area(self): int {
    return self.w * self.h
}
```

## Обобщенное программирование (Generics)

Juno поддерживает работу с обобщенными типами через механизм **мономорфизации**. Это означает, что для каждой уникальной комбинации типов дженерика компилятор генерирует отдельную нативную функцию.

### Обобщенные функции
```swift
fn swap<T>(a: ptr, b: ptr) {
    let temp: T = *a
    *a = *b
    *b = temp
}
```

### Обобщенные структуры
```swift
struct Box<T> {
    let value: T
}
```
При использовании `Box<int>` компилятор создаст структуру с 8-байтным полем. Для `Box<u8>` будет создана версия с 1-байтным полем.
