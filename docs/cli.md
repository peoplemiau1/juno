# CLI и использование

Утилита `bin/juno` является основным интерфейсом для компиляции и запуска программ.

## Основные флаги

*   `bin/juno <file.juno>`: Компиляция файла в `build/output` (по умолчанию x86-64 Linux).
*   `-r`, `--run`: Автоматический запуск программы после успешной компиляции.
*   `-a`, `--arch ARCH`: Выбор целевой архитектуры (`x86_64`, `aarch64`).
*   `-o`, `--output FILE`: Указание пути и имени выходного исполняемого файла.
*   `-t`, `--target OS`: Выбор целевого формата (`linux`, `flat`).

## Примеры использования

```bash
# Компиляция и запуск
./bin/juno -r main.juno

# Сборка для Raspberry Pi (AArch64)
./bin/juno -a aarch64 -o kernel.elf main.juno

# Сборка чистого бинарного образа (Flat Binary)
./bin/juno -t flat -o payload.bin main.juno
```

## Выходные форматы

- **Linux (ELF64)**: Статический исполняемый файл. По умолчанию точка входа — `main`. Не требует наличия динамических библиотек.
- **Flat (Binary)**: Файл, содержащий только машинный код. Идеально подходит для написания ядер ОС или шелл-кодов. Начальный адрес (base RVA) для плоского формата по умолчанию — `0x1000`.
