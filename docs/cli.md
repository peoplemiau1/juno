# CLI и Сборка

## Основные команды
Используйте скрипт `juno` для управления проектом.

*   `juno run <file.juno>`: Быстрая компиляция и запуск.
*   `juno build <file.juno> [-o name]`: Сборка исполняемого файла.
*   `juno new <name>`: Создание шаблона нового проекта.
*   `juno test`: Запуск тестов компилятора.
*   `juno help`: Справка по командам.

## Опции
*   `--hell`: Включить режим обфускации.

## Выходные форматы
*   **Linux (ELF64)**: Стандартный формат для Linux.
*   **Windows (PE64)**: Базовая поддержка исполняемых файлов Windows.
*   **Flat (Binary)**: Чистый машинный код без заголовков (для встраиваемых систем).
    *   Используется через `main_flat.rb`.
