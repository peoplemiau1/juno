# Синтаксис Juno

## Переменные и Типы
В Juno поддерживаются 64-битные целые числа и структуры.

```juno
# Объявление переменной
x = 10
```

## Структуры (Structs)
Структуры позволяют группировать данные.

```juno
struct Point {
    x
    y
}

# Использование
p = Point
p.x = 100
p.y = 200
```

## Функции
Функции определяются через ключевое слово `fn`.

```juno
fn main() {
    output("Hello Juno")
}
```

## Управляющие конструкции
### Условия (If/Else)
```juno
if x == 10 {
    output("Ten")
} else {
    output("Not ten")
}
```

### Циклы (While)
```juno
while x > 0 {
    output_int(x)
    x = x - 1
}
```

## Функции ввода-вывода (Встроенные)
- `output("string")`: Вывод строки в консоль.
- `output_int(var)`: Вывод значения переменной (числа) в консоль.
- `output_member_int(p.x)`: Вывод целочисленного поля структуры.
