# Синтаксис языка Juno

## Основные типы данных
Juno поддерживает следующие встроенные типы:
*   `int`: 64-битное целое число со знаком.
*   `ptr`: Указатель (64-битный адрес).
*   `str`: Строка (фактически указатель на массив байтов, оканчивающийся нулем).
*   `bool`: Логический тип (1 байт, 0 - false, не 0 - true).

## Типы фиксированного размера
Для низкоуровневой работы доступны типы:
*   `i8`, `u8`: 8-битные целые (со знаком и без).
*   `i16`, `u16`: 16-битные целые.
*   `i32`, `u32`: 32-битные целые.
*   `i64`, `u64`: 64-битные целые.

## Переменные
Объявление переменных выполняется с помощью ключевого слова `let` (опционально) или простым присваиванием.
```juno
x = 10
let y: i32 = 20
let ptr: ptr = &x
```

## Управляющие конструкции
### Условия
Скобки вокруг условия опциональны.
```juno
if x > 5 {
    prints("Больше 5")
} else {
    prints("Меньше или равно 5")
}
```

### Циклы
Поддерживаются циклы `while` и `for`.
```juno
while x < 100 {
    x = x + 1
}

for (i = 0; i < 10; i++) {
    print(i)
}
```
