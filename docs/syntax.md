# Спецификация синтаксиса Juno

Juno — статически типизированный язык с явным объявлением типов. Синтаксис спроектирован для обеспечения максимальной читаемости при сохранении прямого отображения конструкций языка в машинные инструкции.

## Система типов

### Примитивные типы

| Тип | Описание | Размер |
| :--- | :--- | :--- |
| `int` | Целое число со знаком (архитектурно-зависимое, обычно 64 бит) | 8 байт |
| `float` | Число с плавающей точкой (IEEE 754) | 8 байт |
| `ptr` | Универсальный указатель | 8 байт |
| `bool` | Логический тип (true/false) | 1 байт |
| `str` | Указатель на последовательность байтов (C-style string) | 8 байт |

### Типы фиксированного размера
Для системных задач доступны типы со строго определенной разрядностью:
*   Целые со знаком: `i8`, `i16`, `i32`, `i64`.
*   Целые без знака: `u8`, `u16`, `u32`, `u64`.

## Объявление переменных и констант

Используется ключевое слово `let`. По умолчанию переменные иммутабельны.

```juno
let x = 42                  // Автоматический вывод типа (int)
let y: u32 = 100            // Явное указание типа
let mut counter = 0         // Изменяемая переменная (mut)
let f: float = 3.14         // Число с плавающей точкой
```

> **Важное замечание**: Переменные должны быть инициализированы при объявлении. Использование неинициализированных данных на уровне языка запрещено.

## Управляющие конструкции

### Условные операторы
Поддерживается стандартная иерархия `if`, `elif`, `else`.

```juno
if condition {
    // выполнение
} elif other_condition {
    // альтернатива
} else {
    // по умолчанию
}
```

### Циклы
Доступны `while` и классический `for`.

```juno
// Бесконечный цикл или условие
while true {
    if stop { break }
}

// Итерационный цикл
for (let i = 0; i < 10; i++) {
    print(i)
}
```

## Структуры данных (Structs)

Структуры определяют компоновку данных в памяти. Поля располагаются последовательно.

```juno
struct Point {
    let x: int
    let y: int
}

// Методы структур
fn Point.distance(self): float {
    // Логика расчета
}
```

> **Заметка для разработчика**: Методы в Juno являются синтаксическим сахаром над обычными функциями, где первый аргумент `self` является указателем на структуру.

## Работа с указателями

Juno предоставляет полный доступ к адресной арифметике.

```juno
let x = 10
let p = &x                  // Получение адреса (pointer to int)
let val = *p                // Разыменование

let buffer = malloc(1024)   // Динамическая память
free(buffer)
```

## Модульная система

Для разделения кода используются ключевые слова `import` и `from`.

```juno
import "std/io.juno"        // Импорт всего модуля
from "std/math.juno" import sin, cos // Импорт конкретных символов
```
