# Синтаксис Juno

Язык Juno использует строгую статическую типизацию и современный синтаксис.

## Типы данных

### Базовые типы
- `int`: 64-битное целое со знаком (стандартный тип).
- `ptr`: Универсальный 64-битный указатель.
- `str`: Указатель на null-terminated строку байтов.
- `bool`: Логический тип (1 байт).

### Типы фиксированного размера
Для системного программирования доступны типы: `i8`, `u8`, `i16`, `u16`, `i32`, `u32`, `i64`, `u64`.

## Объявление переменных

Для объявления используется `let`. Переменные могут иметь явное указание типа.

```swift
let x = 10                  // Вывод типа (int)
let y: u32 = 0xFF           // Явный тип
let mut counter = 0         // Изменяемая переменная
```

## Управляющие конструкции

### Условия (if)
Круглые скобки вокруг условия опциональны.

```swift
if x > 0 {
    print(1)
} elif x < 0 {
    print(-1)
} else {
    print(0)
}
```

### Циклы (while, for)
```swift
while x < 10 {
    x = x + 1
}

for (i = 0; i < 10; i++) {
    print(i)
}
```

## Структуры (Structs)

Структуры определяют пользовательские типы данных. Поддерживаются методы.

```swift
struct Vector {
    let x: int
    let y: int
}

fn Vector.move(self, dx, dy) {
    self.x = self.x + dx
    self.y = self.y + dy
}
```

## Указатели и Память

Juno позволяет работать с памятью напрямую.

```swift
let p = malloc(8)         // Выделение 8 байт
*p = 42                   // Запись по указателю
let val = *p              // Чтение по указателю
free(p)                   // Освобождение
```

Указательная арифметика:
```swift
let p2 = ptr_add(p, 1)    // Смещение на 8 байт (размер элемента)
```
