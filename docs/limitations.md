# Известные проблемы и Ограничения

## Известные ошибки
Исправлены в текущей версии.

## Что не работает (Roadmap)
- **Строки в переменных**: Сейчас строки поддерживаются только как аргументы в `output()`. Сохранить строку в переменную нельзя.
- **Вложенные структуры**: Присваивание одной структуры другой целиком (копирование памяти) пока не реализовано. Только по полям.
- **Массивы**: Нет поддержки синтаксиса `x[10]`.
- **Аргументы функций**: Функции пока не принимают аргументы (`fn name(a, b)`). Все переменные должны быть локальными или через глобальный контекст.
- **float/double**: Поддерживаются только целые знаковые числа `int64`.

## Особенности сборки
- **RWX Sections**: Секция кода помечена как изменяемая. Это может быть помечено антивирусами как подозрительное поведение (Self-modifying code).
