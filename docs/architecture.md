# Архитектура компилятора Juno

## Конвейер компиляции
1.  **Лексер (`lexer.rb`)**: Разбивает исходный текст на токены.
2.  **Парсер (`parser.rb`)**: Строит абстрактное синтаксическое дерево (AST).
3.  **Импортер (`importer.rb`)**: Резолвит зависимости и объединяет файлы.
4.  **Мономорфизатор (`monomorphizer.rb`)**: Создает конкретные версии generic-функций и структур.
5.  **Оптимизатор (`optimizer/turbo.rb`)**: Применяет трансформации AST для повышения эффективности.
6.  **Генератор нативного кода (`codegen/native_generator.rb`)**: Преобразует AST в машинный код x86-64.
7.  **Линкер (`codegen/parts/linker.rb`)**: Формирует структуру исполняемого файла (ELF, PE).

## Особенности реализации
Компилятор полностью автономен и не требует внешних инструментов для сборки. Весь процесс генерации машинного кода реализован на Ruby вручную, включая формирование заголовков ELF и PE.
