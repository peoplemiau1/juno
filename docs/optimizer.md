# Turbo Optimizer

## Цели оптимизатора
Turbo Optimizer предназначен для минимизации накладных расходов и достижения производительности, сравнимой с C и Rust.

## Основные фазы
1.  **Инлайнинг функций**: Функции размером менее 10 нод встраиваются непосредственно в место вызова.
2.  **Развертка циклов**: Циклы с количеством итераций менее 8 разворачиваются для исключения переходов.
3.  **Common Subexpression Elimination (CSE)**: Удаление повторяющихся вычислений и переиспользование результатов.
4.  **Strength Reduction**: Замена тяжелых операций более легкими (например, умножение на 2 заменяется сдвигом влево `<< 1`).
5.  **Константная пропагация**: Вычисление выражений с известными константами во время компиляции.
6.  **Алгебраические упрощения**: Например, `x - x` заменяется на `0`.

## Регистровая аллокация
Juno использует 6 регистров общего назначения для хранения переменных: `RBX`, `R10`, `R12`, `R13`, `R14`, `R15`. Это значительно сокращает количество обращений к стеку.
