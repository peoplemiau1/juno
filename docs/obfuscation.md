# Hell Mode (Обфускация)

## Назначение
Режим Hell предназначен для защиты бинарного файла от реверс-инжиниринга. Включается флагом `--hell`.

## Возможности
*   **Мусорный код**: Генерация бесполезных инструкций, которые затрудняют чтение кода.
*   **Шифрование строк**: Все строковые литералы шифруются и расшифровываются только в момент использования.
*   **Anti-debug**: Встраивание проверок на наличие отладчика (например, через `ptrace`).
*   **Полиморфные инструкции**: Генератор кода может выбирать разные эквивалентные последовательности инструкций для выполнения одной и той же задачи.
*   **Opaque Predicates**: Сложные условия, результат которых всегда известен во время компиляции, но не очевиден для декомпилятора.
