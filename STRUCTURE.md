# Структура проекта Juno

## Корневые файлы
- `main_native.rb` - Компилятор для Windows (PE)
- `main_linux.rb` - Компилятор для Linux (ELF)
- `README.md` - Основная документация
- `docs.md` - Полная документация API

## Директории

### `/src` - Исходный код компилятора
- `lexer.rb` - Лексический анализатор
- `parser.rb` - Синтаксический анализатор
- `codegen/` - Генерация кода
  - `parts/` - Модули генерации
    - `calls.rb` - Диспетчер вызовов функций
    - `logic.rb` - Логические операции
    - `builtins/` - Встроенные функции
- `native/` - PE/ELF сборщики
- `optimizer/` - Оптимизатор кода

### `/stdlib` - Стандартная библиотека
- `std.juno` - Основные функции

### `/tests` - Тесты
- `test_*.juno` - Тестовые файлы

### `/examples` - Примеры программ
- `hello.juno` - Hello World
- `game.juno` - Игра
- `demo_v1.juno` - Демонстрация v1.0
- `demo_v1.1.juno` - Демонстрация v1.1

### `/build` - Артефакты сборки
- `*.exe` - Скомпилированные Windows программы
- `*.obj` - Объектные файлы
- `*.asm` - Ассемблерный код
- `*.c` - Промежуточный C код
- `output_linux` - Linux исполняемые файлы

### `/docs` - Дополнительная документация
- Руководства
- Туториалы
- Спецификации
