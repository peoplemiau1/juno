# Проект Juno: Обзор и выводы

## 1. Общее описание
**Juno** — это современный компилируемый язык программирования, ориентированный на системное программирование под Linux x86-64 и Windows. Язык сочетает в себе простой синтаксис (схожий с Python и C) и высокую производительность благодаря генерации нативного машинного кода.

## 2. Архитектура компилятора
*   **Язык реализации**: Компилятор написан на Ruby.
*   **Генерация кода**: В отличие от многих учебных языков, Juno не использует LLVM или GCC в качестве бэкенда. Он содержит собственный эмиттер машинного кода (x86-64) и построители исполняемых файлов (ELF для Linux, PE для Windows, а также Flat-бинарники для bare-metal).
*   **Оптимизация**: Включает "Turbo Optimizer", поддерживающий инлайнинг функций, развёртку циклов, удаление общих подвыражений (CSE), strength reduction и константную пропагацию.

## 3. Ключевые возможности языка
*   **Синтаксис**: Гибкий синтаксис для определения функций (`fn`, `func`, `def`), опциональные скобки в условиях.
*   **Типизация и структуры**: Поддержка пользовательских структур (`struct`), методов для них, а также обобщенного программирования (Generics) через мономорфизацию.
*   **Низкоуровневый доступ**: Прямая работа с указателями (`&`, `*`), управление памятью (`malloc`, `free`, `mmap`), встроенная поддержка системных вызовов Linux.
*   **Многопоточность**: Поддержка создания потоков, атомарных операций и спин-локов.
*   **Безопасность и защита**: Наличие "Hell Mode" — режима обфускации, который включает шифрование строк, анти-отладочные приёмы (ptrace) и генерацию мусорного кода.

## 4. Стандартная библиотека и экосистема
*   **Сеть**: Встроенная поддержка TCP-сокетов и HTTP-запросов (`curl_get`, `curl_post`).
*   **I/O и Файлы**: Обширный API для работы с файловой системой, включая такие функции, как `file_read_all` и `memfd_create`.
*   **CLI**: Удобный интерфейс командной строки (`juno run`, `juno build`, `juno test`, `juno new`).

## 5. Тестирование и стабильность
*   Проект содержит обширную базу тестов (около 60 тестов) и более 30 рабочих примеров (сортировки, серверы, боты).
*   Большинство базовых функций работают стабильно. Обнаружены некоторые проблемы со стабильностью в сложных сценариях работы с памятью (возможные Segmentation faults), что ожидаемо для проекта такого уровня сложности.
*   Замечена небольшая ошибка в скрипте `test.sh`, где отсутствует установка прав на выполнение для скомпилированных файлов (исправлено в `run_juno_tests.sh`).

## 6. Итоговые выводы
Juno — это мощный инструмент для системного программиста-исследователя. Его главные достоинства:
1.  **Автономность**: Не требует внешних ассемблеров или линкеров.
2.  **Прозрачность**: Код компилятора на Ruby легко читаем и расширяем.
3.  **Богатый функционал**: Поддержка современных сетевых протоколов и многопоточности «из коробки».

Проект отлично подходит для разработки системных утилит, изучения архитектуры ЭВМ и создания легковесных высокопроизводительных приложений.
