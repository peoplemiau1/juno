# Juno: Системный язык программирования

Juno — это низкоуровневый компилируемый язык программирования, разработанный для системного уровня. Язык ориентирован на достижение максимальной производительности, эффективное управление ресурсами и прямой контроль над аппаратным обеспечением. Компилятор генерирует нативный машинный код для архитектур x86-64 и AArch64 (Linux).

## Технические спецификации

*   **Генерация кода**: Прямой перевод AST в машинные инструкции. Компилятор не использует промежуточные зависимости вроде LLVM или GCC, реализуя собственные бэкенды для целевых архитектур.
*   **Управление памятью**: Гибридная модель. Ручное управление (malloc/free) дополняется статическим анализатором (Resource Auditor), который выявляет утечки на этапе компиляции.
*   **Регистровая оптимизация**: Использование алгоритма Linear Scan Register Allocation для эффективного распределения переменных по регистрам процессора.
*   **Система типов**: Строгая статическая типизация. Поддержка дженериков через мономорфизацию (кодогенерация специализированных версий функций для каждого типа).
*   **ABI**: Полное соответствие System V ABI, что обеспечивает бесшовную совместимость с системными вызовами Linux и существующими библиотеками на языке C.

## Быстрый старт

### Требования
Для работы компилятора необходим интерпретатор Ruby версии 3.0 или выше.

### Компиляция и запуск
```bash
# Простая компиляция и запуск примера
./bin/juno -r examples/hello.juno
```

### Параметры CLI
```text
Usage: juno [options] <input_file>
    -a, --arch ARCH        Целевая архитектура (x86_64, aarch64)
    -o, --output FILE      Путь к выходному бинарному файлу
    -t, --target OS        Целевая ОС (linux, flat)
    -r, --run              Запуск бинарного файла после успешной компиляции
```

## Краткий обзор синтаксиса

Juno совмещает синтаксическую выразительность современных языков с семантической прозрачностью C.

```juno
import "std/io.juno"

struct Node {
    let value: int
    let next: ptr
}

fn main(): int {
    // Выделение памяти под структуру
    let head: Node = malloc(16)
    head.value = 100
    
    // Работа с float (IEEE 754)
    let ratio: float = 0.75
    
    print(head.value)
    
    // Статический аудит требует явного освобождения
    free(head)
    return 0
}
```

> **Примечание**: Resource Auditor автоматически проверяет, что каждый вызов `malloc` или `open` имеет соответствующий `free` или `close`. Нарушение этого правила вызовет предупреждение компилятора.
